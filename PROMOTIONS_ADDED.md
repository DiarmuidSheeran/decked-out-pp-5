## ğŸ”§ Manage Promotions

### Add More Promotions
1. Go to [Admin â†’ Products](https://8000--019a6575-edee-73a7-859a-1c29738a3f20.eu-central-1-01.gitpod.dev/admin/products/product/)
2. Click on a product
3. Check "Is on promotion"
4. Set "Promotion price" (lower than regular price)
5. Optionally check "Clearance" for deep discounts
6. Save

### Mark New Arrivals
1. Edit product in admin
2. Check "New arrival"
3. Save

### Remove Promotions
1. Edit product
2. Uncheck "Is on promotion"
3. Uncheck "Clearance"
4. Save

### Bulk Actions
Select multiple products in admin and use bulk actions to:
- Mark as promotion
- Mark as clearance
- Mark as new arrival
